Definition section

An array is a data structure which is used to store the multiple elements of same data types.

An array elements is stored in contiguous memory location and it is contiguous i.e. one after another.

An array element can be accessed using it's index and it starts from 0 to length-1.

Advantage

  • Memory efficient
  • Fast access time
  • Ease to use
  • Suitability for statis data

Disadvantage

  • Fixed size
  • Inefficient insertion/deletion
  • Memory allocation issue
  • Homogeneous data only

Example

  • A list of students in a classroom: The students might be seated in a specific order (e.g., alphabetically, or by height). Each student has a position in that ordered sequence.
  • A row of books on a shelf: Each book has a specific place on the shelf. The first book is at the beginning, the second next to it, and so on.
